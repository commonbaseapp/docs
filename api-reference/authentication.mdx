---
title: "Authentication"
description: "Authenticating Commonbase requests."
---

Authenticate Commonbase requests by providing your workspace API key in the
`Authorization` header.

```bash
'Authorization': '<api_key>'
```

### Permissions

API Keys are scoped to a single Commonbase workspace.
An API Key provides access to submit invocations for every project
inside the workspace it is associated with.
